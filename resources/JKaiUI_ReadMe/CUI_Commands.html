・CUIコマンドの説明


1, 記号の説明

    ()内はユーザーが任意の値にする必要があります。

    []内はオプションです。省略可能です。

    | = or


2, 各コマンドの説明


  1.     %(command) [(commands)]

  2.         コマンドは行頭の半角の%から始まります。行頭に%にない場合はコマンドとみなされません。commandには以下    のすべてのコマンドが利用できます。

  3.         commandsにはsend, ask以外のコマンドが利用できます。


  5.     send (commands)

  6.         commandsで指定したコマンドで表示される結果をチャットとして送信できます。commandsには以下のverからopenconfigまで指定できます。


  8.     ask (commands2)

  9.         commands2で指定したコマンドを相手に実行してもらった結果をチャットで送信してもらいます。commands2ではver, diag, congfigが利用できます。


  11.     ver

  12.         JKaiUIのバージョンを表示します。


  14.     help

  15.         CUIコマンドのヘルプを表示する予定です。現状はコマンドのリストを表示するのみです。

  16.        

  17.     show friends|users|rooms|bookmarks

  18.         現在のオンラインフレンドリスト、現在のいるアリーナのユーザー、現在いるアリーナの部屋リスト、ブックマークリストをそれぞれ表示します。


  20.     phrase #(num)

  21.         定型文を表示します。numは定型文リストの上からの番号です。


  23.     emoti #(num)

  24.         顔文字を入力欄に表示します。numは顔文字番号です。顔文字パネルの左上が１で右に向かって１ずつ増えていきます。


  26.     setconfig (ConfigTag) (value)

  27.         config値を設定します。ReadMeの最後に説明しているplistのTypeがConfigTagになります。値の種類は３種類あります。これもplistで確認する必要があります。

  28.             boolean型：true, falseのみ

  29.             int型：整数値のみ

  30.             String型：文字列


  32.     config all|original|other|log|chatpm|roomuser|sound

  33.         現在の設定値を表示できます。allなどはカテゴリです。


  35.     diag all|server|network|reachable|hardware|nic|lib|engine|ver|platform

  36.         diagモードでみれる情報を表示できます。allなどはカテゴリです。


  38.     go parent|#(num)|(relative_address)

  39.         現在いるアリーナを基点として移動します。parent, #0を指定すると一つ上のアリーナに移動します。#(num)を指定すると表示されている部屋の上からnum番目に移動します。

  40.         relative_addressを指定するとその名前の部屋に移動します。


  42.     gofull full_address

  43.         フルアドレスを指定して移動します。（例：gofull Arena/PSP/Action）


  45.     arenapm #(num)

  46.         arenapmを開きます。表示されている上からnum番目のユーザーのPMを開きます。


  48.     follow #(num)

  49.         フレリストのオンラインになっているnum番目のユーザーの現在いる場所に移動します。


  51.     pm #(num)

  52.         フレリストのオンラインになっているnum番目のユーザーのPMを開きます。


  54.     bookmark #(num)

  55.         ブックマークリストのnum番目の場所に飛びます。


  57.     chatreset

  58.         チャット画面のリセット。


  60.     chatlock

  61.         チャットのロック。


  63.     cahtunlock

  64.         チャットのアンロック。


  66.     logwindow

  67.         ログウィンドウを表示、非表示にする。


  69.     exit

  70.         JKaiUIを終了する。


  72.     connect

  73.         kaiengineに接続する。


  75.     disconnect

  76.         kaiengineとの接続を切る。


  78.     messengermode

  79.         メッセンジャーモードにする。


  81.     arenamode

  82.         アリーナモードにする。


  84.     diagmode

  85.         Diagモードにする。

  86. 　

  87.     create [(description)] [(maxplayernum)][(password)]

  88.         部屋を作成する。オプションをすべて省略した場合、各値を入力するウィンドウが表示される。

  89.             maxplayernum, passwordを指定しなかった場合、それぞれ４、なしで部屋が作られる。

  90.             passwordを省略した場合、パスワードなしで部屋が作成される。

  91.             他の省略パターンはできない（必ず１つめのオプションはdescriptionと解釈される。2つめ、3つめも同様）。


  93.     openconfig

  94.         設定画面を開く。
